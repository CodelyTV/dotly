## Installation

Using curl
```bash
bash <(curl -s https://raw.githubusercontent.com/CodelyTV/dotly/master/installer)
```

Using wget
```bash
bash <(wget -qO- https://raw.githubusercontent.com/CodelyTV/dotly/master/installer)
```
